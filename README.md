# type-single-page

## About

I'm a graphic design major but I'm also minoring in illustration. I'm specifically very intersted in comics.
That's why I chose the 'Indoor Kid' font for this website.

## Process

Going into the assignment, I wanted to keep my pre-established color scheme. However, I changed it a bit to 
fit this assignment better. I made my mockup in Figma and transferred the assests to Illustrator to export the 
individual images as svgs. I wish I just did the whole thing in Illustrator. In Figma, it's easier to see what 
it will look like as a site but it's harder to get the assests. Most of the text was just the 'Indoor Kid' font
but for the history section, I used 'Happy Monkey' so the paragraph would be easier to read. I wanted the fonts 
to be interesting but also readable.  I'm dyslexic so I wanted to fully make sure it was easily readble.
I also didn't want the site to be overwhelming to look at. 

## Challenges

I've found that having dyslexia makes coding very difficult. It's harder to read the lines of code and find 
where everything is. Starting out with the base code from Lara was difficult because I had no familiarity with it 
and it was harder to figure out what line of code did what. I also found it difficult to make the site adaptable 
to go inbetween a desktop and a mobile layout. Overall, I was able to get it to work for a desktop but the mobile 
layout is debatable.
